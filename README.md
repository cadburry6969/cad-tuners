# Tuner cars display and sell

### qb-core/shared Configuration
- Add `tuner` job
- If want the vehicles to be shown in catelogue then add `['shop'] = 'tuner'`
- If want the vehVicles to be sold/spawn then add `['category'] = 'tuners'`
